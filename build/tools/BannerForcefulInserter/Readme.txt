=============================================================
バナー無理矢理挿入ツール
=============================================================


NITROアプリのマスタリングでは、バナーのハッシュを計算する必要があるため、
バナーデータがROM内に存在しない場合、正常にマスタリングできません。

本ツールでは、バナーが存在しないROMに無理矢理バナーを挿入して、
マスタリングできるROMを作成します。


使用方法
===============

バッチファイルにNITROアプリのROM(SRL形式)をドラッグアンドドロップしてください。


TWLアプリの場合
==================

バナーの有無はマスタリングと無関係です。
そのため、本ツールは、TWLアプリ非対応です。
TWLアプリが入力された場合、エラーを表示して処理を中断します。
