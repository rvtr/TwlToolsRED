○バージョンアップ
  前のバージョンが存在し、ショップでアップデートさせたい場合には、propertiesファイルを更新してはいけません。
  この場合、前のバージョンのCLS処理で使用した properties ファイルが必要となります。
  必ずCLS処理の前に、 properties ファイルがどのバージョンのものか確認してください。

○パラメータ
  アプリの用途しだいでは properties ファイルの中身を変更する必要があります。

  ・TITLE_TYPE  [TWL_APP/TWL_GAME/TWL_SYS]
    ユーザアプリの場合、通常 "TWL_APP" ですが、
    ショップでレーティング表示をしたいときには "TWL_GAME" に変更する必要があります。
    システムアプリの場合、通常 "TWL_SYS" となっており、変更する必要はありません。

  ・ALLOW_COMMON_TICKET  [1/0]
    ユーザアプリの場合、通常 "0" ですが、
    ショップ以外からインポートしたいときには(TwlNMenuなど)、"1" に変更する必要があります。
    システムアプリ場合、通常 "1" となっており、変更する必要はありません。
