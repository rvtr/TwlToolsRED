はじめに
=======================================================================

このパッケージには、システムアプリ版TWLマスターエディタとよばれる、
TWLシステムアプリのマスターROM提出確認書を作成するためのツールが含まれています。

=======================================================================
システムアプリ版マスターエディタと通常版との違い
=======================================================================

システムアプリが入力可能であること以外には、
通常版と挙動は大きく異なることはありませんが、以下の違いがあります。

  * リージョンで「全リージョン」が選択できます。
    (RSFファイルで全リージョンを指定していたとしても、
     エラーとならないので、ご注意ください。)

　* エラーのチェック項目が一部追加されます。
